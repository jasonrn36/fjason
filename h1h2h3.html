<h1>Tag de Título de página</h1>
Éssa tag é utilizada para formatar o título principal de uma página
como o título de um carderno de jornal ou machete.

<h2>Tag de Título Texto principal</h2>
Já a tag H2 tem seu uso para formatar o título do texto de uma página,
ou blocos de textos.

<h3>Tag de Título Texto principal</h3>
Outra tag usada para formatar o título de blocos de textos é a H3

<h4>Tag de Título Texto principal</h4>
Já a tag H4 pode ser usada para descrições de blocos de textos ou menus.
<h5>Tag de Título Texto principal</h5>
A tag H5 pode ser utilizada para descrições de textos e fotos.
<h6>Tag de Título Texto principal</h6>
Já a tag H6 pode ser utilizada apenas para descrições pelo seu tamanho e formato muito pequeno.
