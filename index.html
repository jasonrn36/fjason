Aprendendo HTML5, CSS e JavaScript
Por Jason Ramalho Nascimento
Atualizado em 12-05-2024
Olá Mundo! Seja bem vindo ao site onde você irá aprender muito sobre HTML5 e CSS.
Para quem é da área da infor­mática é sempre bom estar atua­lizado nessa área.­ Melhor ainda é saber Passo-a-passo como se faz cada coisa. E foi justa­mente para isso que este site foi criado, mesmo que no futuro este conteúdo esteja desatua­lizado, a forma como são feitas as forma­tações de páginas em HTML e CSS será sem­pre válido, por que aqui será ensi­nado a SE­MÂN­TICA. A semân­tica em resumo é uti­li­zada para dar melhor orga­ni­zação as páginas e apli­cações web. Nos nave­gadores e dispo­si­tivos atuais, isso é muito impor­tante para dar ordem e saber onde encon­trar cada parte do código da página na hora da manu­tenção.
